# Serum File Sorter Organizer
Uma ferramenta de automação em Python projetada para reorganizar bibliotecas enormes de presets do Serum. Ela escaneia diretórios recursivamente, identifica tipos de instrumentos com base em padrões de nomes de arquivos (Baixo, Lead, Pad, etc.) e os consolida em uma estrutura de pastas organizada e categorizada, sem excluir os arquivos originais.
